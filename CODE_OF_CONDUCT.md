# Code of Conduct

ZIAN-Projekte sollen professionell, respektvoll und konstruktiv bleiben.

Erwartet werden:

- respektvolle Kommunikation
- sachliche Kritik
- klare Dokumentation
- verantwortungsvoller Umgang mit Daten, Assets und Quellcode

Nicht akzeptiert werden persönliche Angriffe, diskriminierende Inhalte oder missbräuchliche Nutzung der bereitgestellten Ressourcen.
