# Accessibility

Auch ein dunkles Premium-Design muss gut lesbar bleiben.

## Regeln

- Fließtext nicht unter 16 px setzen
- ausreichenden Kontrast verwenden
- Aura-Effekte nicht hinter langen Texten platzieren
- Fokuszustände sichtbar gestalten
- Buttons und Links klar unterscheiden

## Textfarben auf dunklem Grund

- Headlines: `#F6F6EE`
- Fließtext: `#CCCCCC`
- Sekundärtext: `#888888`
- Linien: `rgba(246,246,238,0.12)` bis `rgba(246,246,238,0.28)`
