# GitHub-Integration

## Empfohlene Repositories

```text
.github
zian-brand-system
```

## Organisationsprofil

Das Repository `.github` enthält:

```text
profile/README.md
```

Diese Datei dient als öffentliche Übersicht der Organisation.

## Brand-System

Das Repository `zian-brand-system` enthält alle Corporate-Design-Dateien.

## Einzelne Projekt-Repositories

Projekt-Repositories sollen nur referenzieren:

```md
## Brand

This project follows the ZIAN Brand System.
```

So bleibt das Corporate Design zentral wartbar.
