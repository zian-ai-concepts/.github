# Bildsprache

## Passend

- abstrakte 3D-Formen
- Metall, Stein, Carbon, Glas
- Schatten, Lichtkanten, Reflexionen
- technische Interfaces
- Architektur, Systeme, Module
- dunkle Projektvisualisierungen

## Nicht passend

- bunte Stock-Illustrationen
- verspielte Icons
- Pastellflächen
- typische Business-Stockfotos
- zu viele Effekte gleichzeitig

## Prinzip

Die Bildwelt soll Tiefe und Präzision zeigen. Sie wirkt hochwertig, ruhig und kontrolliert.
