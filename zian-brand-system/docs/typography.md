# Typografie

## Display

Empfohlen: Space Grotesk oder Sora.

Einsatz: Headlines, Hero-Texte, starke Titel.

## Body

Empfohlen: Inter oder IBM Plex Sans.

Einsatz: Fließtext, Dokumentation, UI-Texte.

## Mono

Empfohlen: JetBrains Mono oder IBM Plex Mono.

Einsatz: Labels, Projektcodes, technische Kennzeichnungen, Datenpunkte.

## Beispiel

```text
PROJECT_04 / SYSTEM DESIGN / ACTIVE
```

## Stil

- Headlines großzügig setzen
- Labels in Uppercase mit Letterspacing
- Text nicht zu klein auf dunklem Grund verwenden
