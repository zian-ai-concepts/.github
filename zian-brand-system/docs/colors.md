# Farben

Das Farbsystem bleibt nah am Signet: dunkle Basis, metallische Zwischenwerte und ein kühler Aura-Akzent.

| Name | Hex | Verwendung |
|---|---:|---|
| Void Black | `#000000` | Haupt-Hintergrund, maximale Tiefe |
| Obsidian | `#050505` | Website- und App-Hintergruende |
| Deep Graphite | `#0A0A0A` | Sektionen, Cards, Navigation |
| Carbon Grey | `#222222` | Panels und sekundäre Flächen |
| Iron Grey | `#2F2F2F` | Schattenflächen, Hover |
| Steel Grey | `#555555` | Linien, dezente Icons |
| Titanium | `#888888` | Sekundärtext, Rahmen |
| Silver Mist | `#CCCCCC` | Fließtext auf dunklem Grund |
| Monolith White | `#F6F6EE` | Headlines, Highlights |
| Aura Silver | `#E8EEE2` | Glow, aktive Zustände |
| Soft Alloy | `#DDE4DA` | Dezente Aura und Lichtschein |

## Verhältnis

- 80 Prozent dunkle Basis
- 15 Prozent metallische Grauwerte
- 5 Prozent Aura-Akzent

## Do

- große, ruhige dunkle Flächen verwenden
- Akzentfarbe sehr sparsam einsetzen
- Textkontraste prüfen

## Don't

- keine bunten Zusatzfarben ohne Systemgrund
- keine Pastellflächen
- keine grellen Neonfarben
