# Brand-Checklist

## Vor Veröffentlichung prüfen

- Logo nicht verzerrt
- ausreichend Schutzraum
- Farbwerte aus Tokens verwendet
- keine unpassenden Zusatzfarben
- Typografie konsistent
- dunkle Flächen nicht überladen
- Text gut lesbar
- Repository-README folgt ZIAN-Stil
- Social Preview vorhanden
- Projekt referenziert das Brand-System

## Schnellbewertung

Ein Asset passt zu ZIAN, wenn es diese Fragen beantwortet:

1. Wirkt es präzise?
2. Wirkt es ruhig?
3. Wirkt es technisch?
4. Hat es ausreichend Raum?
5. Ist der Aura-Akzent sparsam eingesetzt?
